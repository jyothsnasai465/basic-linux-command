# Basic Linux Commands:
- cd (change directory)
- pwd (present working Directory)
- ls (list of files)
- rm (remove)
- mkdir (create a directory or folder)
- rmdir (remove directory)
- touch (create a new file)
- nano (edit a file)
- cat (file description)
## clone the repository
- git clone repositorylink
## Initialise the git
- git init
